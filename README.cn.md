
# TLDR-Pages 重新打包发布目标

本项目旨在重新打包和发布 [TLDR-Pages](https://github.com/tldr-pages/tldr)，以优化特定使用场景下的体验。

## 使用场景

假设 `tldr` 模块在以下场景中使用：

1. **有限的网络带宽**：需要快速下载和查阅命令手册
2. **中国地区网络环境**：确保在中国大陆网络环境下可正常访问

## 设计目标

基于上述使用场景，我们制定了以下设计目标：

1. **按需下载语言包**：将各个语言包分开打包，支持按需下载。考虑到大多数用户通常只使用一种语言，这能显著减少初始下载体积。
2. **压缩打包格式**：采用 `xz` 格式进行压缩打包，进一步缩小下载体积约 30%。
3. **国内镜像托管**：将打包后的资源托管在 [Gitee](https://gitee.com) 上，确保国内用户能够快速、稳定地访问。

## 实现方式

- 每个语言包独立打包，用户可根据需要选择下载
- 使用 `xz` 压缩算法，在保持内容完整性的同时最大限度减小文件大小
- 所有资源同步至 Gitee 镜像，以方便加速

通过以上优化，我们期望为用户提供更轻量、更快速、更可靠的 `tldr` 使用体验。
